Self-hosted privacy


Secure software for self-hosted privacy

We focus on software that can be hosted on a single server or virtual machine, at home or at a cloud provider.

tlsproxy

A simple but powerful TLS proxy, reverse proxy, and web server. It provides TLS encryption for any number of TCP or HTTP servers, and any number of server names concurrently on the same port.

It can also be used as a Reverse Proxy for HTTP(S) services, and control access to these services with user authentication and authorization.

photos

A photo album with state-of-the-art end-to-end encryption. Store and share files with friends and family securely using your own server.

Only you and those with whom you share your files can access their content. Even the server can't decrypt your data.
